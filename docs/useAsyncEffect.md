## About
A version of useEffect that accepts an async function


[//]: # (Main)

## Installation

```
npm install rooks
```

## Importing the hook

```javascript
import {useAsyncEffect} from "rooks"
```

## Usage

```jsx
function Demo() {
  useAsyncEffect();
  return null
}

render(<Demo/>)
```
